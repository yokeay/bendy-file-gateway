## 2. WASM Bridge

Go зј–иҜ‘дёә WASM еҗҺиҝҗиЎҢеңЁ JS е®ҝдё»дёӯгҖӮеҸҢеҗ‘йҖҡдҝЎпјҡ

**Go вҶ’ JS (exports)**:
- `handleRequest(method, path, headers, body, remoteAddr) вҶ’ (status, headers, body)` вҖ” HTTP иҜ·жұӮдё»е…ҘеҸЈ
- `ready()` вҖ” еҲқе§ӢеҢ–е®ҢжҲҗдҝЎеҸ·

**JS вҶ’ Go (imports)**:
- `dbQuery(sql, params) вҶ’ JSON` вҖ” жҹҘиҜў
- `dbExec(sql, params) вҶ’ JSON` вҖ” еҸҳжӣҙ
- `cacheGet(key) вҶ’ (value, found)` вҖ” зј“еӯҳиҜ»еҸ–
- `cacheSet(key, value, ttl)` вҖ” зј“еӯҳеҶҷе…Ҙ
- `cacheDel(key)` вҖ” зј“еӯҳеҲ йҷӨ

## 3. иҜ·жұӮз”ҹе‘Ҫе‘Ёжңҹ

```
1. Platform ж”¶еҲ° HTTP иҜ·жұӮ
2. Platform еәҸеҲ—еҢ–иҜ·жұӮдёәеӯ—иҠӮ
3. Platform и°ғз”Ё WASM.handleRequest()
4. Go дёӯй—ҙд»¶й“ҫеӨ„зҗҶпјҡ
   a. Recovery вҖ” жҚ•иҺ· panic
   b. CORS вҖ” и·ЁеҹҹеӨ„зҗҶ
   c. Logging вҖ” иҜ·жұӮж—Ҙеҝ—
   d. Auth вҖ” иә«д»ҪйӘҢиҜҒпјҲз§ҹжҲ· HMAC жҲ–з®ЎзҗҶе‘ҳ Sessionпјү
   e. Quota вҖ” й…ҚйўқжЈҖжҹҘпјҲд»…з§ҹжҲ·иҜ·жұӮпјү
   f. Router вҖ” и·Ҝз”ұеҲҶеҸ‘
   g. Handler вҖ” дёҡеҠЎйҖ»иҫ‘
5. Handler йҖҡиҝҮ WASM imports жҹҘиҜў/жӣҙж–°ж•°жҚ®еә“е’Ңзј“еӯҳ
6. Handler йҖҡиҝҮ HTTP е®ўжҲ·з«Ҝи°ғз”ЁеӯҳеӮЁеҗҺз«Ҝ
7. Go иҝ”еӣһе“Қеә”з»ҷ Platform
8. Platform еҸҚеәҸеҲ—еҢ–е№¶иҝ”еӣһ HTTP е“Қеә”
```

## 4. еҲҶж”Ҝзӯ–з•Ҙ

| еҲҶж”Ҝ | еҶ…е®№ | ж•°жҚ®еә“ | зј“еӯҳ | е…ҘеҸЈ |
|------|------|--------|------|------|
| `main` | е…ұдә« Go + React жәҗз Ғ | Schema ж–Үд»¶ | ж—  | зәҜеә“д»Јз Ғ |
